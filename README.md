# CoffeeTrove .NET Client

.NET client library for accessing CoffeeTrove coffee data.

## Overview

CoffeeTrove is a comprehensive coffee platform with 440,000+ cafes worldwide, brewing guides, bean profiles, and origin data.

- **Homepage**: [https://coffeetrove.com](https://coffeetrove.com)
- **Documentation**: [https://coffeetrove.com/docs](https://coffeetrove.com/docs)
- **Map**: [https://coffeetrove.com/map](https://coffeetrove.com/map)

## Installation

```bash
dotnet add package CoffeeTrove
```

## Quick Start

```csharp
using CoffeeTrove;

Console.WriteLine($"CoffeeTrove v{Info.Version}");
Console.WriteLine($"API Base: {Info.BaseUrl}");
```

## Data Coverage

- 440,000+ cafes worldwide
- Brewing methods and guides
- Coffee bean profiles
- Origin country data
- Golden Drop scoring system
